# COVID-19 Data Exploration — SQL Portfolio Project

## Project Overview
Exploratory data analysis of global COVID-19 data using SQL Server.
The dataset covers cases, deaths, and vaccinations from January 2020 onwards,
sourced from Our World in Data.

## Skills & Concepts Used
- Joins
- CTEs (Common Table Expressions)
- Temp Tables
- Window Functions (PARTITION BY)
- Aggregate Functions
- Converting Data Types
- Creating Views

## Dataset
- Source: Our World in Data (ourworldindata.org/covid-deaths)
- Tables: Covid_Deaths, CovidVaccinations

## Key Queries Performed
1. Death percentage per country (likelihood of dying if infected)
2. Percentage of population infected per country
3. Countries with highest infection rates
4. Countries with highest death counts
5. Continent-wise death breakdown
6. Global total cases vs deaths
7. Rolling vaccination count using window functions
8. Percentage of population vaccinated (via CTE and Temp Table)
9. Created a View for future Tableau visualizations
